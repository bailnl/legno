
## Input props

| 参数 | 说明 | 类型 | 默认值 | 可选值 |
| --- | --- | --- | --- | --- |
| value | 输入框内容 | String | "" |  |
| type | 输入框类型 | String | "text" | "password" |
| composition | 是否感知输入法 | Boolean | false | - |
| select-on-focus | 当focus时选中内容 | Boolean | false | - |

input支持`原生属性`所有属性
